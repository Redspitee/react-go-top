## Install

```
npm i react-go-top
or
yarn add react-go-top
```

## Usage

```
import Gotop from 'react-go-top';

render(){
  return(
    <Gotop />
  )
}
```

## API

imgSrc              string    default
style               object    default 
target              object    default window
visibilityHeight    number    default 400
